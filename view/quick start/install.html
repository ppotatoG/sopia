다운로드

소피아는 가장 최신 버전을 사용하는 걸 권장하고 있습니다
최신 버전을 다운로드 해 주세요.

  • 1. 홈페이지에서 다운로드

    홈페이지는 항상 최신의 파일만 관리하고 있기 때문에 페이지에서 다운받는 것을 권장합니다.

    다운로드 링크

  • 2. 구글드라이브에서 다운로드

    구글 드라이브는 과거 버전부터 현재까지의 모든 버전을 모아두고 있습니다.

    다운로드 링크

소피아 실행

소피아는 PC에서만 실행시킬 수 있습니다. Windows OS 만 사용가능합니다.
모바일, Mac OS는 사용 불가합니다.
